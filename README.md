# CodeAlpha Tasks

A collection of web development projects built with HTML, CSS, and JavaScript.

## 📋 Projects

### 1. Age Calculator
A simple and interactive age calculator that computes your exact age in years, months, and days based on your date of birth.

**Features:**
- Calculate age from date of birth
- Display results in years, months, and days
- Clean and responsive user interface
- Built with pure HTML, CSS, and JavaScript

### 2. TodoList
A task management application to help you organize your daily activities and boost productivity.

**Features:**
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Persistent storage using localStorage
- Responsive design

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required!

### Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/manikanta-devs/codealpha_tasks.git
   ```

2. Navigate to the project folder:
   ```bash
   cd codealpha_tasks
   ```

3. Open the desired project folder (AgeCalculator or TodoList)

4. Open `index.html` in your web browser
